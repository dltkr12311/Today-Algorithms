# Multiplication Table

- 오늘은 너무나도 간단한 구구단을 하였다. 작성할때 너무도 쉬워 놀라웠다 역시 직접 작성해봐야 한다.
  작성을 하지 않으니 막연하게만 느껴졌던거 같다. 다시 한번 꺠닫는다...

  ```js
  for (let i = 2; i <= 9; i++) {
    console.log(`${i}단`);
    for (let j = 1; j <= 9; j++) {
      console.log(`${i} * ${j} = ${i * j}`);
    }
  }
  ```
